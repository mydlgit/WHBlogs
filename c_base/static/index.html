About Static

静态与动态

 静态类型: 对象的静态类型指其在程序中被声明时所采用的类型; 动态类型则是指“目前所指对象类型”, 可在运行期发生改变;
 考虑带有缺省参数的virtual函数: virtual函数是动态绑定，缺省参数是静态绑定, 意味着你可能在"调用一个定义于derived class 内的虚函数，但使用base class提供的缺省参数值"

函数中的静态变量

 当变量声明为static时，空间将在程序的生命周期内分配。即使多次调用该函数，静态变量的空间也只分配一次，前一次调用中的变量值通过下一次函数调用传递。这对于在C / C ++或需要存储先前函数状态的任何其他应用程序非常有用。

静态成员函数

 其不能访问非静态成员变量，因为静态成员函数属于类，通过类访问，调用函数时没有提供this指针, 所以其不能访问非静态成员变量（因为其属于某个具体的对象）。
